*** Main parameters used with all run options
1@ Pedigree file path
1# /input_directory/pedigree_file.txt

2@ SNV genomic positions file path 
2# /input_directory/SNV_markers_genomic_positions.txt

3@ SNV genotypes file path
3# /input_directory/SNV_markers_genotypes.txt

4@ Linkage region boundaries, cM
4# left_boundary_number right_boundary_number

5@ Position of linkage marker with maxLOD score, cM
5# number

6@ Output directory
6# /output_directory

*** Additional parameters used only with "haplotyping" or "full" run options
7@ Linkage markers genomic positions file path
7# /input_directory/linkage_markers_genomic_positions.txt

8@ Meiosis indicators file path
8# /input_directory/meiosis_indicators.mi

9@ Number of iterations in meiosis indicators file
9# number

*** Additional parameters used only with "haplosharing" run option
10@ Haplotype sequences file path
10# /input_directory/haplotype_sequences.txt

11@ Core set of cases file path
11# /input_directory/core_set_of_cases.txt

*** Optional parameter
12@ Seed number (default is 1234)
12#
